How GVI is Calculated

The Green View Index (GVI) measures the percentage of vegetation visible in a street-level image. A deep learning model (semantic segmentation) identifies different objects in each image. The green-colored pixels represent detected vegetation (trees, shrubs, grass). GVI = (green pixels) / (total pixels) √ó 100%
